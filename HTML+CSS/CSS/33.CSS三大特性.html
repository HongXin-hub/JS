CSS三大特性

1、层叠性 2、继承性 3、优先级
我是一行描述层叠的文本
层叠性： 1.样式冲突，遵循就近原则。哪个样式距离结构近，就执行哪个样式。 2.样式不冲突，不会层叠。

我是一行描述继承的文本

继承性： 1.书写CSS样式表时，子标签会继承父标签的某些样式，如文本颜色和字号。想要设置一个可继承的属性，只需将它应用于父元素即可。 注意：恰当地使用继承可以简化代码，降低CSS样式的复杂性。（text-，font-，line- 这些元素开头的都可以继承，以及color属性）
王者荣耀
优先级： 1、继承或者* 0, 0, 0, 0 2、元素或者标签 0, 0, 0, 1 3、类或者伪类 0, 0, 1, 0 4、id 0, 1, 0, 0 5、行内样式 1, 0, 0, 0 6、!important 无穷大 权重是优先级的算法，层叠是优先级的表现
