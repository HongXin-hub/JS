HTML 表单用于搜集不同类型的用户输入。
在HTML中，一个完整的表单通常由 表单域、提示信息、表单控件 3部分构成。
表单控件：
包含了具体的表单功能项，如单行文本输入框、密码输入框、复选框、提交按钮、重置按钮等。
提示信息：
一个表单中通常还需要包含一些说明性的文字，提示用户进行填写和操作。
表单域：
它相当于一个容器，用来容纳所有的表单控件和提示信息。如果不定义表单域，表单中的数据就无法传送到后台服务器。

input 控件

type属性： text 单行文本输入框 password 密码输入框 radio 单选按钮 checkbox 复选框 button 普通按钮 submit 提交按钮 reset 重置按钮 image 图像形式的提交按钮 file 文件域 name属性：控件的名称 value属性：input控件中的默认文本值 size属性：input控件在页面中的显示宽度 checked属性：checked定义选择控件默认被选中的项 maxlength属性：控件允许输入的最多字符数
