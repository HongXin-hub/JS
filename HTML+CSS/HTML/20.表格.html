border 设置表格的边框（像素值，默认border="0"无边框）
cellspacing 设置单元格与单元格边框之间的间距（像素值，默认2px）
cellpadding 设置单元格内容与单元格边框之间的间距（像素值，默认1px）
width 设置表格的宽度（像素值，）
height 设置表格的高度（像素值，）
align 设置表格在网页中的水平对齐方式（left、center、right）
